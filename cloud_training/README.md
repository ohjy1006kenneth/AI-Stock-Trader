# cloud_training

Cloud-side research and training code.

This folder contains the heavy AI / research path for the project:
- feature generation
- dataset building
- predictive model training
- validation / backtesting
- Hugging Face Space packaging for cloud-side jobs

Key subareas:
- `data_pipelines/` — build aligned training features from news, market, and context inputs
- `model_architecture/` — predictive model code and policy contracts
- `backtesting/` — validation and promotion metrics
- `training/` — training / export / pipeline entrypoints

This is the main place for model-development work.
