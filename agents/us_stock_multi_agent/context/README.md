# Context Handoff Files

These Markdown files are the persistence layer between specialized agents.

- `macro_watchlist.md` → output from Macro Scout
- `technical_signals.md` → output from Technical Analyst
- `risk_decisions.md` → output from Portfolio Risk Manager
- `execution_report.md` → output from Mock Execution Agent
- `daily_summary.md` → end-of-day portfolio summary

Review `macro_watchlist.md` before enabling the downstream buy pipeline.
